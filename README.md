## Customer Churn Prediction

### Problem Statement
Customer churn is a major challenge for banks and subscription-based businesses. This project aims to predict whether a customer is likely to leave the bank based on demographic and account-related features.

### Dataset
The dataset contains customer information such as credit score, age, balance, tenure, activity status, and geography.

### Approach
- Data cleaning and encoding
- Feature selection
- Train-test split
- Random Forest classification
- Model evaluation using confusion matrix, classification report, and accuracy
- Feature importance analysis

### Results
The model successfully identified key churn drivers, with customer age, balance, and activity level emerging as the most influential factors.

### Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
